# 控件名称熟悉
| 控件名字 | 含义 | 
| -|-|
| objectName | 控件名称 |
| geometry | 相对坐标系 |
| sizePolicy | 控件大小策略 | 
| minimumSize | 最小尺寸 |
| maxinumSize | 最大尺寸 |
| font | 字体 |
| cursor | 光标 |
| windowTitle | 窗口标题 |
| windowIcon | 窗口图标 |
| iconSize | 图标尺寸 |
| statusTip | 任务提示栏 |
| text | 控件文本 |
| shortcut | 快捷键 |
